# 第○章　使役構文（〜させる）

## 1. 使役とは

使役構文とは、「〜に…させる」「〜を…される」などのように、**他者に動作を行わせる**意味を表す構文です。

フランス語では主に **faire + 不定詞** の形で使役を表現します。

---

## 2. 基本構造

フランス語における使役構文の基本構造は以下のとおりです：

> **構文：**  
> `[主語] + [faire（活用形）] + [動作主（目的語）] + [動詞の原形]`

**意味：** 「（主語が）〜に…させる」

---

## 3. faire の活用（現在形）

| 人称 | 活用 |
|------|------|
| je   | fais |
| tu   | fais |
| il / elle / on | fait |
| nous | faisons |
| vous | faites |
| ils / elles | font |

---

## 4. 例文

| 文 | 意味 |
|----|------|
| Je fais réparer ma voiture. | 私は車を修理させる。／修理してもらう。 |
| Il fait venir un technicien. | 彼は技術者を呼ばせる。 |
| Elle fait laver les vitres. | 彼女は窓ガラスを洗わせる。 |
| Nous faisons travailler les élèves. | 私たちは生徒たちに勉強させている。 |

※ 「faire + 不定詞」は「誰がその動作をするか（動作主）」を文中に明示する必要があります（省略も可能）。

---

## 5. 使役構文の種類

### 5.1 明示的な使役（誰にさせるかを示す）

> **構文：**  
> `[主語] + faire + [動作主（目的語）] + [動詞の不定詞]`

例：
- Le professeur **fait lire** les élèves.  
  （先生は生徒に読ませる）

- Je **fais réparer** mon vélo par un mécanicien.  
  （私は自転車を修理してもらう）

### 5.2 動作主を省略した構文

> **構文：**  
> `[主語] + faire + [動詞の不定詞]`

例：
- Elle **fait préparer** un gâteau.  
  （彼女はケーキを作らせる／作ってもらう）

この場合、**誰がその動作をするのかは明示されません**。文脈から判断する必要があります。

---

## 6. 間接受け身的な使役

使役構文は「〜してもらう」という **受け身的使役** にもなります。

例：
- Je fais couper mes cheveux.  
  → 「私は髪を切ってもらう」  
  （＝自分で切るのではなく美容師などにやってもらう）

- Elle fait nettoyer sa robe.  
  → 「彼女はドレスをクリーニングしてもらう」

※ この用法は特に日常表現でよく使われます。

---

## 7. 語順の注意

使役構文では、**人称代名詞**や**目的語代名詞**が間に入る位置に注意する必要があります。

例：
- Je **le fais venir**.  
  （私は彼を来させる）→ 代名詞 le は faire の直前に置く。

- Elle **me fait rire**.  
  （彼女は私を笑わせる）

---

## 8. よく使われる表現

| 表現 | 意味 |
|------|------|
| faire entrer | 入らせる |
| faire sortir | 出させる |
| faire attendre | 待たせる |
| faire savoir | 知らせる |
| faire comprendre | 理解させる |

例：
- Je vais **te faire savoir** la décision.  
  （私は君にその決定を知らせるつもりだ）

---

## 9. まとめ

- フランス語の使役構文は、**faire + 不定詞** の形で「〜に…させる」「〜を…される」を表す。
- **動作主**を明示することで、誰にその行動をさせるかが分かる。
- 動作主を省略した場合は、**受け身的な使役（〜してもらう）**にもなる。
- 目的語代名詞の位置や活用形に注意。
