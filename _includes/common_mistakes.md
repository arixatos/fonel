# よくある誤りと正しい対応 ✅

| 誤り         | 正しい対応         | 備考 |
|--------------|--------------------|------|
| ø → ø        | ø → **ö**          | 狭い丸唇前舌母音 |
| [ʒ] → ž      | [ʒ] → **j**        | žは使用しない |
| /k/ → k      | /k/ → **c**        | Fonel語では k 非使用 |
| [ɔ] → o      | [ɔ] → **ò**        | 広いoにはアクサン・グラーヴ |
| [ə] → 無視   | [ə] → **ë**        | 文法的に重要（fémina ŝwa）|
| [g] → j      | [g] → **g**        | g→jは不可、ʒ→j |
