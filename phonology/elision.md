# 正規形と脱落形

フォネル語の単語には、**正規形（fòrm règülyèr）と脱落形（fòrm rédüit）**の2種類が存在します。  
脱落形とは、**語末の発音が省略された形**であり、特に**語末子音**や**曖昧母音（ëなど）**が省略される傾向があります。  
このように語末音が発音されない現象そのものを、**「脱落（ŝütë fonic）」**と呼びます。


1. **正規形（脱落可能子音を発音）**  
   - 発音（IPA）：[pəˈtit]  
   - 解釈：脱落可能子音 [t] を保持して発音する形。Fonelの基本形。  
   - 表記例：`pëtit`

2. **脱落形（脱落可能子音を発音しない）**  
   - 発音（IPA）：[pəˈti]  
   - 解釈：脱落可能子音 [t] を脱落させた形。Fonelの脱落形。  
   - 表記例：`pëti'`

---

#### 辞書上の扱い（例）  
- pëtit (adj.) : 小さい  
  - 正規形: [pəˈtit]  
  - 脱落形: [pəˈti] （次が子音の場合など）




---

## 1. 脱落形が使われる条件

脱落形は、以下のような**音韻的または文脈的条件**のもとで使用が認められます：

- **句末**など、後続の語がなく、語が単独で発音される場合  
- 後続語との間に**連続母音**や**連続子音**が生じ、発音しにくくなる場合

このような場面では、音の流れを滑らかにする目的で脱落が許容されます。

---

## 2. 正書法における原則と例外

フォネル語の正書法では、**原則として正規形（脱落しない形）を用いる**ことが推奨されています。  
ただし、以下のような場合には、**脱落形の表記が許容される、あるいは望ましいとされることもあります**：

- **口語表現の引用**（話し言葉を忠実に再現したい場合）  
- **詩的・音韻的な調整**（音節数や韻律の都合による省略）  
- **発音学習の支援**（実際の発音と綴りの対応を示す場合）


## 4. 脱落は義務ではない

脱落は文法的な必須要素ではなく、**発音上の都合に応じて話し手が任意に選択**できます。  
そのため、**文語・朗読・強調**など、正確な音形を保ちたい状況では、あえて脱落せず正規形で発音することも可能です。

# 脱落形（フォルム・レデュイ）

脱落形（フォルム・レデュイ）とは、語末の発音が省略された形であり、特に語末子音や曖昧母音（ëなど）が脱落する傾向があります。  
また、語末鼻音の脱落に伴う鼻母音化など、語と語が連結する際の音声変化も含まれます。  
本稿では、これらの脱落形について詳しく説明し、分類・例示を行います。

---

## 1. 脱落形の概要

脱落形とは、語末の発音が省略される現象を指します。  
Fonel語では特に語末の子音や曖昧母音（ë）が脱落しやすく、これが文中での発話の流暢さや連結の自然さに寄与しています。

例を挙げると、`tak bat` が `ta’ bat` に変化するケースが典型です。  
このように、語末の子音 `k` が脱落し、アポストロフィ（’）で脱落を示します。

---

## 3. 脱落の仕組みと分類

### 3.1. 基本的な脱落

語末の脱落可能な子音や曖昧母音が単独で脱落する現象です。

- 例：  
  - `tak ` → `ta’`  
    （/tak/ → /ta/：語末の /k/ が脱落し、アポストロフィで示されます）  
  - `dua` → `du’`  
    （/dua/ → /duʔ/：語末の曖昧母音「a」が脱落）

## 脱落のパターン

多くの言語では、母音同士が接すると融合や省略が、子音と母音の境界では連結が起こるのが一般的です。  
特にフランス語はその傾向が顕著です。  
フランス語にルーツを持つFonelでは、その特徴に伴う文法を強制ルールではなく選択可能なオプションルールとして引き継いでいます。



### 連結の基本原則

フォネル語では、語と語が続くと**原則として自然に連結されて発音**されます。  
このとき、語末音と語頭音の組み合わせは以下の4通りに分類されます：

| パターン | 内容             | 発音の自然さ | 備考                     |
|----------|------------------|----------------|----------------------------|
| C + V    | 子音 + 母音      | 非常に自然     | 問題なし                   |
| V + C    | 母音 + 子音      | 非常に自然     | 問題なし（後続子音と接続） |
| C + C    | 子音 + 子音      | やや困難       | 前の語の子音が脱落しやすい |
| V + V    | 母音 + 母音      | やや困難       | 前の語の母音が脱落しやすい |

> ※ C = 子音, V = 母音

### 脱落の基本パターン

- **C + C** の場合：語末の子音が脱落し、単一の子音に  
- **V + V** の場合：語末の母音が脱落し、単一の母音に

| 状況     | 脱落しやすい語末音 | 結果的な構造 |
|----------|--------------------|----------------|
| C + C    | 子音               | C（単一化）     |
| V + V    | 母音               | V（単一化）     |

### 例

| 正規形   | 脱落形    | 音の変化                     | 意味（仮）     |
|----------|-----------|------------------------------|----------------|
| tak bat  | ta’ bat   | /tak bat/ → /ta bat/         | 「〜は打つ」  |
| dua ota  | du’ ota   | /dua ota/ → /du ota/         | 「二つの家」  |

---




### 子音連結脱落（コンシェット）


語末の脱落可能子音が直後の語の母音に連結しながら脱落する現象です。  
Fonel語の連結脱落は、語の連結を滑らかにし、発音しやすくする役割があります。

- 例：  
  - `tak atta` → `ta’ atta`  
  - `bak el` → `ba’ el`


一般的な脱落可能子音は以下の通りです。

| 脱落可能子音 | IPA  | 備考                                               |
|--------------|------|----------------------------------------------------|
| -t           | [t]  |                                                    |
| -n           | [n]  | 鼻母音の脱落形としても扱う（例：bon [bɔ̃] ↔ [bɔn]）|
| -r           | [ʁ]  |                                                    |
| -z           | [z]  |                                                    |



#### n連結脱落（ナシュット）

語末の /n/ がする際、語の母音が鼻母音化する現象です。子音連結脱落（コンシェット）に含まれます。



典型的な例として、`bon` の発音変異があります。  
この語には、基本形（正規形）として母音＋`n` の連結形 [bɔn]（Fonel表記: **bon**）が存在し、単独や文末など特定の環境で **n脱落に伴い鼻母音化** した [bɔ̃]（Fonel表記: **bõ**）となる脱落形が見られます。

| 形態         | 発音（IPA） | Fonel表記 | 説明                         |
|--------------|-------------|-----------|------------------------------|
| 正規形 | [bɔn]       | bon       | 母音＋`n` の連結形（正規形） |
| 脱落形 | [bɔ̃]       | bõ        | **n脱落に伴う鼻母音化**による脱落形         |

### 例：`bon ami`（良い友達）

- 正規形（連結）：`[bɔ.n a.mi]` → **bon ami**  
- 脱落形（単独）：`[bɔ̃]` → **bõ**



---
##＃ 母音結合：語末母音の脱落  
**Ŝütë dë la vwayèl final**

Fonelでは、**フランス語において語末母音（特に `e`）が必ず脱落（エリジオン）する場面でも、それを保持して発音することができます**。

Fonelでは、こうした「フランス語であればエリジオンする場合」に限り、**脱落形が使用可能**です。  
また、この脱落形の使用は**義務ではなく任意**です。

## 4. 連結脱落の詳細
### 連結の基本原則

フォネル語では、語と語が続くと**原則として自然に連結されて発音**されます。  
このとき、語末音と語頭音の組み合わせは以下の4通りに分類されます：

| パターン | 内容             | 発音の自然さ | 備考                     |
|----------|------------------|----------------|----------------------------|
| C + V    | 子音 + 母音      | 非常に自然     | 問題なし                   |
| V + C    | 母音 + 子音      | 非常に自然     | 問題なし（後続子音と接続） |
| C + C    | 子音 + 子音      | やや困難       | 前の語の子音が脱落しやすい |
| V + V    | 母音 + 母音      | やや困難       | 前の語の母音が脱落しやすい |

> ※ C = 子音, V = 母音

### 脱落の基本パターン

- **C + C** の場合：語末の子音が脱落し、単一の子音に  
- **V + V** の場合：語末の母音が脱落し、単一の母音に

| 状況     | 脱落しやすい語末音 | 結果的な構造 |
|----------|--------------------|----------------|
| C + C    | 子音               | C（単一化）     |
| V + V    | 母音               | V（単一化）     |

### 例

| 正規形   | 脱落形    | 音の変化                     | 意味（仮）     |
|----------|-----------|------------------------------|----------------|
| tak bat  | ta’ bat   | /tak bat/ → /ta bat/         | 「〜は打つ」  |
| dua ota  | du’ ota   | /dua ota/ → /du ota/         | 「二つの家」  |

連結脱落では、脱落形が単に語末音が消えるだけでなく、脱落音が直後の語の音に影響を与えます。  
これにより、音声的に滑らかな連結が実現されます。

- コンシェットの例：  
  - `tak bat` → `ta’ bat`  
    /k/ が脱落し、語末にアポストロフィを挿入。  
- ナシュットの例：  
  - `pan ota` → `pã ota`  
    /n/ が脱落し鼻母音化。

---

## 脱落の表記方法

発音上の脱落を反映させたいとき、フォネル語では以下のような表記を行います：

### 表記原則

- 脱落した音は**書かず**、その位置に **アポストロフ（’）** を置く。  
- アポストロフは、「この位置の音が省略された」ことを示す記号であり、**脱落音の種類までは明示しない**。

### 使用される文脈

- 会話スクリプト  
- 発音ガイドや音声教材  
- 詩や歌詞など、音のリズムが重要な文体

### 表記例

| 意味       | 正規形    | 脱落形     | 発音変化                  |
|------------|-----------|-------------|----------------------------|
| 「彼は行く」 | il va     | i’ va       | /il va/ → /i va/           |
| 「三つの目」 | tri oku   | tr’ oku     | /tri oku/ → /tr oku/       |




Fonelの正書法では、正規形発音を文字転写した正規形記法が基本であり、公的な文書や公式な文章では常に正規形を用いることが推奨されます。

脱落形発音を文字転写しただけの脱落形記法は日常的な簡易記法として用いられることがありますが、正書法上は正しくありません。

一方、脱落形の後ろにアポストロフ（'）を付加して脱落した音があることを示す省略形記法は、主に会話文や口語表現など、音の省略を視覚的に正確に再現

---

## 6. まとめ

Fonel語における脱落形は、語末の子音・曖昧母音が連結の影響で省略されることによって生じます。  
これらは語の連結を滑らかにし、発話の自然さを保つために不可欠な音声変化です。  
特に連結脱落（コンシェット）や n連結脱落（ナシュット）はFonel語の特徴的な現象であり、正規形から脱落形への変換規則として理解しておく必要があります。
